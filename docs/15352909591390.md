# 谁在贩卖你的隐私
小编最近在在别人共享的某盘里发现一个文件，文件名叫”手持证件文件.rar“，这个压缩文件有6.30G之大。基于这个文件名，小编猜到这有可能是关系到大量的信息泄露，于是就下载下来看个究竟。
耗费3个多小时，下载下来，解压打开文件之后，小编彻底震惊了。这已经不是信息泄露那么简单了，这里面居然有着完完整整的5071个人的身份证正反面以及真人手持身份证照片。
